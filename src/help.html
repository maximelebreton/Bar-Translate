Bar Translate's help & tips

Get started

  1. Jump to address bar
  2. Type t, and press space or tab
  3. And get the translation as you type! (the default language is your browser language)

Target a specific language

  • Prefix your query with any ISO language code and a space, like en to start translating in English

Specify source language

  • You can use the > operator to specify source and target, like fr>en to start translating from French into English

Examples

  • en bonjour => "hello" (English)
  • es hello=> "holà" (Spanish)
  • zh hello=> "你好" (Chinese simplified)
  • en>es four => "cuatro" (from English into Spanish)
  • fr>es four => "horno" (from French into Spanish)
  • hello => "bonjour" (takes the browser default language, French in my case)

How to find these language ISO codes?

Just start typing the English name of any language, and you will get suggestions

Shortcuts

Jump to the address bar

  • Windows: Ctrl + L or Alt + D or F6
  • Mac: ⌘ + L
